= @ink-feather-org/ts-utils

This repository contains a plugin for typedoc to support ascii doc readme files.
To use this plugin specify the `readme` option for typedoc.

== tsconfig.json

[source, json]
----
{
  "typedocOptions": {
    "entryPoints": ["src/index.ts"],
    "out": "docs",
    "readme": "README.adoc",
    "excludePrivate": true
  }
}
----
